Create the multiple ajax posts list/grid view widgets and content block shortcodes with various ajax pagination options.

== Description ==

Ajax load more post is the best wordpress plugin to create list and grid view of the posts and custom posts using ajax load more pagination, next-previous links or numeric pages. It has fully customizable and responsive for templates and allows to hide certain categories posts.  

This plugin supports unlimited shortcodes and widget for the home and sub-pages content blocks and sidebar using advanced shortcode generation forms from admin panel.

* Supports ajax load more posts.

== Installation ==

1. Upload and extract the zip file downloaded to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'APLM' page from dashboard and create new shortcode to add into any page content (pro version only)
4. Configure widgets from Appearance > Widgets and assign to page sidebar.
